# Vue 라이프사이클 (Lifecycle)

Vue 인스턴스가 생성되고 → 화면에 나타나고 → 업데이트되고 → 사라지기까지의 과정.
각 단계에 "훅(hook)"을 걸어서 원하는 코드를 실행할 수 있다.

---

## 전체 흐름도

```
    createApp({...}).mount('#app')  ← 내가 실행
            │
    ┌───────▼───────┐
    │  beforeCreate  │  data, methods 등록 전
    │                │  → 거의 안 씀
    └───────┬───────┘
            │
    ┌───────▼───────┐
    │    created     │  data, methods 사용 가능
    │                │  → 초기 데이터 가공
    │                │  → API 호출 (DOM 불필요 시)
    └───────┬───────┘
            │
    ┌───────▼───────┐
    │  beforeMount   │  HTML 렌더링 직전
    │                │  → 거의 안 씀
    └───────┬───────┘
            │
    ┌───────▼───────┐
    │   mounted  ★   │  HTML 렌더링 완료
    │                │  → API 호출
    │                │  → DOM 접근
    │                │  → setInterval 시작
    │                │  → 외부 라이브러리 초기화
    └───────┬───────┘
            │
     (data 변경 발생)
            │
    ┌───────▼───────┐
    │ beforeUpdate   │  화면 다시 그리기 직전
    │                │  → 업데이트 전 DOM 상태 저장
    └───────┬───────┘
            │
    ┌───────▼───────┐
    │   updated      │  화면 다시 그리기 완료
    │                │  → 업데이트 후 DOM 접근
    │                │  → 주의: 여기서 data 변경하면 무한루프
    └───────┬───────┘
            │
     (앱 종료 / 컴포넌트 제거)
            │
    ┌───────▼───────┐
    │ beforeUnmount  │  제거 직전
    │                │  → 정리 작업 시작
    └───────┬───────┘
            │
    ┌───────▼───────┐
    │  unmounted     │  제거 완료
    │                │  → clearInterval
    │                │  → removeEventListener
    │                │  → WebSocket 연결 해제
    └────────────────┘
```

---

## 자주 쓰는 훅 TOP 3

```
1위: mounted    → 화면 준비 후 실행 (API 호출, DOM 접근)
2위: created    → data 준비 후 실행 (DOM 필요 없는 초기화)
3위: unmounted  → 정리 작업 (타이머, 이벤트 리스너 제거)
```

---

## created vs mounted

```
                created         mounted
─────────────  ──────────────  ──────────────
data 접근       ✅ 가능         ✅ 가능
DOM 접근        ❌ 아직 없음     ✅ 가능
언제 실행        더 빨리         HTML 렌더 후
API 호출        ✅ 가능         ✅ 가능 (보통 여기서)
```

> 헷갈리면 mounted에서 하면 된다. 실무에서 대부분 mounted 사용.
