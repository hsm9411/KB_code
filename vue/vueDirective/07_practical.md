## 패턴 1: 할 일 목록 (Todo)

가장 대표적인 Vue 학습 예제. 거의 모든 핵심이 들어간다.

```
사용되는 개념:
├── data        → 할 일 배열, 입력값
├── v-model     → input 양방향 바인딩
├── v-for       → 목록 렌더링
├── v-if        → 빈 목록 처리
├── @keyup.enter → 키보드 이벤트
├── @click      → 삭제, 토글
├── methods     → 추가, 삭제, 토글
├── computed    → 남은 할 일 개수
├── :class      → 완료 상태 스타일
└── watch       → 로컬스토리지 저장
```

---

## 패턴 2: 검색 + 정렬 테이블

```
사용되는 개념:
├── data        → 원본 데이터, 검색어, 정렬 기준
├── v-model     → 검색어 입력
├── v-for       → 테이블 행 렌더링
├── computed    → 필터링 + 정렬된 결과
├── @click      → 정렬 기준 변경
└── :class      → 현재 정렬 열 표시
```

---

## 패턴 3: 폼 입력 + 유효성 검사

```
사용되는 개념:
├── v-model     → 각 필드 양방향 바인딩
├── computed    → 유효성 검사 결과
├── :disabled   → 유효하지 않으면 버튼 비활성
├── :class      → 에러 필드 표시
├── v-if        → 에러 메시지 조건부 표시
└── @submit.prevent → 폼 제출 처리
```
